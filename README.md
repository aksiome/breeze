# Breeze - A Modern Sphinx Theme

A modern Sphinx theme built with TypeScript and Vite. 

[demo](https://sphinx-breeze-theme.readthedocs.io/en/stable/)


# Setup

pip:
```
pip install sphinx-breeze-theme
```

uv:
```
uv add --dev sphinx-breeze-theme
```

Update ``conf.py``:
```
html_theme = "breeze"
```
